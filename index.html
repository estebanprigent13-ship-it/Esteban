Veille technologique

Cloud & Virtualisation

Suivi des évolutions technologiques autour de l'infrastructure cloud et des solutions de virtualisation, dans le cadre du BTS SIO SISR.

☁️
Pourquoi ce sujet ?

Le Cloud et la virtualisation transforment profondément les infrastructures informatiques des entreprises. En tant que futur technicien SISR, maîtriser ces technologies est devenu indispensable : la quasi-totalité des entreprises migrent leurs services vers des environnements hybrides ou full-cloud, réduisant les coûts et augmentant la flexibilité des systèmes d'information.

🖥️
La Virtualisation

La virtualisation permet de faire tourner plusieurs systèmes d'exploitation sur une seule machine physique grâce à un hyperviseur. On distingue deux types :

  • Type 1 (bare-metal) — VMware ESXi, Microsoft Hyper-V, Proxmox
  • Type 2 (hosted) — VirtualBox, VMware Workstation
Hyper-VProxmoxVMware ESXiVirtualBox
🌐
Le Cloud Computing

Le cloud propose des ressources informatiques à la demande via Internet. Trois modèles de déploiement :

  • Cloud public — mutualisé, géré par un fournisseur tiers (AWS, Azure, GCP)
  • Cloud privé — infrastructure dédiée à une seule organisation
  • Cloud hybride — combinaison des deux, tendance dominante
AWSMicrosoft AzureGoogle CloudOVHcloud
⚙️
Modèles de service

Trois niveaux de service selon la responsabilité partagée :

  • IaaS — Infrastructure as a Service (ex : Azure VM)
  • PaaS — Platform as a Service (ex : Azure App Service)
  • SaaS — Software as a Service (ex : Microsoft 365)
IaaSPaaSSaaS
